# Governance

This repo is governed by four simple rules:

1. Runtime and artifact truth outrank prose.
2. Contradictions stay visible until resolved.
3. Private staging is the default until exact-commit verification is complete.
4. External-license and runtime dependencies do not become silent passes.

Current repo status:

- visibility posture: private staging
- default release posture: no public release
- current blocking class: proof-boundary and verification gaps
