# Support

Use `architects@zer0pa.ai` for:

- private security reports
- licensing questions
- repo-boundary or proof-boundary escalation

Security reporting details live in `SECURITY.md`.

<p>
  <img src="../.github/assets/readme/section-bars/reporting.svg" alt="REPORTING" width="100%">
</p>

When reporting a technical problem, include:

- exact commit SHA
- exact command run
- stdout and stderr
- the file path or proof path in dispute
