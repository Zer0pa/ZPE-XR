# Code Of Conduct

Participation in this repo is expected to stay professional, evidence-led, and non-hostile.

## Expected

- respectful technical disagreement
- clear evidence for claims
- honest handling of negative results

## Unacceptable

- harassment or intimidation
- suppressing failures to protect a narrative
- overstating what the repo proves
- publishing another person's private information

Report conduct issues to `architects@zer0pa.ai`.
