Reports with:

- an exact command
- a precise file or artifact path
- stdout/stderr
- a clear statement of what you expected versus what you observed

<p>
  <img src="../.github/assets/readme/section-bars/what-does-not-get-a-response-here.svg" alt="WHAT DOES NOT GET A RESPONSE HERE" width="100%">

- requests to narrate blockers into passes
- questions already answered in the repo front-door docs
- vague reports without a reproducible path
